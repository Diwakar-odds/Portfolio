## Local Development

1. Clone the repository:
```bash
git clone https://github.com/Diwakar-odds/Portfolio.git
```

2. Navigate to the project directory:
```bash
cd Portfolio
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```
